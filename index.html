Введение в HTML

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц. Он определяет структуру и содержание веб-страницы.

Простой пример HTML

<!DOCTYPE html>
<html>
<head>
    <title>Моя страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это моя первая веб-страница.</p>
</body>
</html>

HTML состоит из тегов, которые обозначаются угловыми скобками < >. Большинство тегов имеют открывающий и закрывающий тег (например, <p> и </p>).

Структура HTML-документа

Каждый HTML-документ имеет стандартную структуру:

  • <!DOCTYPE html> - объявление типа документа
  • <html> - корневой элемент страницы
  • <head> - содержит метаинформацию о странице
  • <body> - содержит видимое содержимое страницы

Основные элементы внутри <head>

<meta charset="UTF-8"> - кодировка символов
<title>Заголовок страницы</title> - заголовок во вкладке браузера
<link rel="stylesheet" href="styles.css"> - подключение CSS
<script src="script.js"></script> - подключение JavaScript

Работа с текстом

HTML предоставляет множество тегов для форматирования текста:

Примеры текстовых тегов

<h1>Заголовок 1 уровня</h1>
<h2>Заголовок 2 уровня</h2>
...
<h6>Заголовок 6 уровня</h6>

<p>Абзац текста</p>

<strong>Важный текст (жирный)</strong>
<em>Акцентированный текст (курсив)</em>

<br> - перенос строки
<hr> - горизонтальная линия

Ссылки

Тег <a> создает гиперссылки:

<a href="https://example.com">Текст ссылки</a>

Атрибуты ссылок

  • href - URL-адрес назначения
  • target - где открыть ссылку (_blank - в новой вкладке)
  • title - всплывающая подсказка

Примеры ссылок

<a href="page.html">Внутренняя ссылка</a>
<a href="https://site.com" target="_blank">Внешняя ссылка</a>
<a href="#section">Якорная ссылка</a>
<a href="mailto:email@example.com">Ссылка на email</a>

Изображения

Тег <img> вставляет изображение на страницу:

<img src="image.jpg" alt="Описание изображения">

Атрибуты изображений

  • src - путь к файлу изображения
  • alt - альтернативный текст (обязателен для доступности)
  • width/height - размеры изображения
  • title - всплывающая подсказка

Пример изображения

<img src="logo.png" alt="Логотип сайта" width="200">

Списки

HTML поддерживает несколько типов списков:

Маркированный список

<ul>
    <li>Пункт 1</li>
    <li>Пункт 2</li>
    <li>Пункт 3</li>
</ul>

Нумерованный список

<ol>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
    <li>Третий пункт</li>
</ol>

Список определений

<dl>
    <dt>HTML</dt>
    <dd>Язык разметки</dd>
    <dt>CSS</dt>
    <dd>Язык стилей</dd>
</dl>
